What is it?

The application "Style Guide" is an online reference for the Information Technology (IT) department at Foxtel.

This guide aims to

  • Ensure correctness and consistency for Customer Systems Internal Applications.
  • Establish standards for clear and consistent mark-up.

If you cannot find what you are looking for, or if you have any additional questions, comments or suggestions for this guide, please contact the Customer Systems UX Team.

Integration

Make sure you build a release version of the style guide first using "grunt dist".


  • CORE
  • RMS

Copy CSS and JS files from the Distribution List folders into corresponding Core folders:

  • CSS files - Found under ...\dist\css\
    1. foxtel.min.css
    2. foxtelHacks.min.css
    3. foxtelIE.min.css

    Into ...\ootbworkspace\web-commons\base\web\include\css\

  • JS files - Found under ...\dist\scripts\
    1. foxtel.min.js
    2. polyfills.min.js

    Into...\ootbworkspace\web-commons\base\web\include\js\
