Defines styles for custom components.

Usage

  • Add the .Breadcrumbs class to support navigation
  • Add the .Or class to create a nice visual separation when requiring input
  • Add the .Alert class to indicate something requires attention
  • Add the .CKH class to indicate something will open in Central Know How
  • Add the <hr> element to define a thematic change in the content
  • Add the .Loading class to indicate the user has to wait
  • Add the .DataInfo, .DataWarning or .DataError class to provide feeback
  • Add the .DataInfoBox, .DataWarningBox or .DataErrorBox class to emphasize feeback
  • Add the .Number class to display a count
  • Add the .Rating class with .C, .P, .G, .PG, .M, .MA, .AV, .R or .NC modifier to display TV Classifications
