Defines styles for info, success, warning and error pop-up messages.

Usage

  • To use message box excute the $.messageBox() jQuery plugin.
  • The plugin accepts the following paramaters:
    • type determines the style of the message box info, success, warning or error
    • title title bar value
    • content content value
    • isCloseButton display close button true or false
    • buttons array, each value contains a name and optional callback { name: 'button name', callback: function () { // operation here }}
