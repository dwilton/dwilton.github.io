Easily create nice looking buttons, that come in different styles.

Usage

  • <button> elements will automatically inherit the button style.
  • Add the .Button class to an <a> element.
  • Add the disabled attribute to a <button> element to disable the button.
  • Add the .Secondary class deemphasizes to look.
  • Add the .PositiveButton or .NegativeButton class to indicate behaviour.
  • Add the .Processing class to indicate the user should wait (requires additional mark-up).
  • Wrap with the .ActionButtons class to increase the buttons size and align to the right.
  • Use the .SegmentedButtons class to create toggle buttons
